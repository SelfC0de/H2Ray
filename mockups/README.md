# H2Ray UI prototype

Интерактивный браузерный прототип Android-приложения `com.h2ray.app`.

## Запуск

Откройте `index.html` в браузере или запустите локальный HTTP-сервер:

```powershell
python -m http.server 8080
```

Команду нужно выполнять из корня проекта. После запуска откройте `http://localhost:8080/mockups/`.

## Реализованные сценарии

- подключение и отключение VPN;
- выбор и поиск профиля;
- проверка задержки;
- редактирование профиля;
- импорт из ссылки, QR-кода, подписки, файла и Xray JSON;
- выбор режима маршрутизации;
- переключатели настроек;
- журнал подключения.

Прототип не создаёт реальное VPN-подключение и не обрабатывает конфигурации Xray.
